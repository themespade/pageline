== PageLine  ==
PageLine is modern style free one page responsive WordPress theme. PageLine can be used for business, portfolio, corporate, agency, photography, freelancers and almost any kind of sites. It is SEO friendly with featured image supports and custom colors. This theme is fully widgetized, so it is easier to operate.Theme customizer can be used to change color, upload logo and other theme settings. You can get quick SUPPORT through support page http://themespade.com/support/ and view live DEMO at http://demo.themespade.com/pageline/
